# 原计划是利用豆瓣的公共API写个资讯类的。后发现方片收集该款应用，个人比较喜欢极简风格的东西。所以大致仿作一个。


## 整体架构
**框架**:目前正在学习理解MVVM模式，利用Rx + databinding 自己粗略搭了个MVVM框架。

由于理解不够透彻，目前View，Model耦合度较高。

数据:主要为笔记工具软件，数据存储采用Sqlite。

**UI**:整体模仿方片收集的UI风格。未自定义控件。大多使用原生控件。
